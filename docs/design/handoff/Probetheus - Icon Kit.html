Iconography

Thin-line constellation glyphs that replace every emoji across the chrome. Geometric, drawn on a single 1.25px stroke, rendered in currentColor so any glyph inherits gold (earned/active) or mist (quiet) from its surface. Built to read cleanly from 12 to 20px.

27 glyphs · 24×24 viewBox · stroke 1.25 · round caps + joins · no fills except node cores

Drawing rules
Keep the set coherent as it grows.
// geometry
Built from circles, hexagons, rhombi and straight runs. Constellation-adjacent — never illustrative.
// stroke
1.25px at 24px. Round caps and joins. Node cores are the only filled element, and only when something is "live."
// color
currentColor only. Emphasis is gold or color — never a heavier stroke, never a glow on a resting icon.
