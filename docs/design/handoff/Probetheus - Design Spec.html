Design Spec

The transcribable layer of Void Premium. The void is the canvas; gold is the stolen fire — it marks only what the player earned and built. A typical screen is ~85% void, ~10% text, ≤5% gold. Everything breathes slightly violet; nothing is pure black or pure white.

Motion grammar

// the traveling spark
Whenever value moves through the network, a spark shows it. Drift, don't snap. Ambient elements move at minutes-per-cycle speeds; pulses breathe on 2–4s cycles. Nothing strobes, nothing shakes by default.
// timing
Easing cubic-bezier(0.4, 0, 0.2, 1) — everywhere.
UI transitions 240–400ms.
Ambient drift minutes / cycle.
Respects prefers-reduced-motion: drift slows, sparks fade in place, shake off.
