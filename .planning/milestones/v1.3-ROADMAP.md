# Milestone v1.3: Signal Distribution System

**Status:** SHIPPED 2026-02-06
**Phases:** 5-10 (plus 8.5)
**Total Plans:** 14

## Overview

This milestone adds sector-specific signal types that are exclusive to designated sector categories (Resource-Rich, Data Haven, Ancient, Asteroid Field). Each sector type spawns unique signals with distinct visuals, enhanced rewards, and behaviors that encourage strategic exploration. The feature extends existing signal generation, rendering, and reward systems through additive patterns rather than architectural changes.

## Milestone Summary

**Decimal Phases:**
- Phase 8.5: Probethium Synthesis (inserted after Phase 8 for exotic material economy)

**Key Decisions:**
- Extension pattern (not new systems) — 95% infrastructure exists, add conditional logic
- 22.5% exclusive spawn rate — base rate targeting 15-30% range with shell bonus headroom
- Exclusive 2x multiplier (not additive) — cleaner reward curve
- Relic minimum uncommon (not rare) — prevents worthless drops
- Exotic Crystal 60/40 split — 60% enhanced exotics, 40% mixed variety
- Resource profiles on discovery — distance-weighted RNG for progression feel
- Profiles independent from types — maximizes sector variety
- Synthesis batch processing — process all available batches at once
- Three-phase synthesis animation — spiral, flash/shake, drift (3 seconds)
- "Open Frequency" Standard sector messaging — positive framing
- Wide tolerance ranges for RNG tests — prevents flaky statistical tests
- Programmatic sector creation in tests — deterministic control, no RNG flakiness

**Issues Resolved:**
- HSL detection before gradient stops (exotic_crystal cycling HSL crashes hexToRgba)
- initializeSector spawn fix (missing spawnDiscoveryBonusSignals call)
- Signal array isolation in tests (prevent cross-test pollution)
- ctx.save()/restore() for particles (isolated canvas state prevents artifacts)

**Issues Deferred:**
- None

**Technical Debt Incurred:**
